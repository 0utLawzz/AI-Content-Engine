# AI CONTENT ENGINE — PHASE 2

Excellent foundation.

Do NOT rewrite the existing architecture.

Extend it.

Treat the current project as the permanent core.

All new features must be added as plugins or services.

---

## PRIORITY 1

Build an AI Orchestrator.

Purpose:

Generate complete content packages using AI providers.

Supported providers:

OpenAI

Gemini

Claude

Future providers

The renderer must never depend directly on any AI provider.

Only the orchestrator communicates with AI.
